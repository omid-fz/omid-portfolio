  PORTFOLIO WEBSITE — README

FILES INCLUDED
--------------
  index.html   — Main HTML structure (all pages)
  style.css    — All styling and responsive design
  script.js    — Page navigation and animations
  README.txt   — This file

HOW TO VIEW LOCALLY
-------------------
1. Keep all four files in the SAME folder.
2. Open index.html in any modern browser (Chrome, Firefox, Safari, Edge).
   Do NOT open style.css or script.js directly — only index.html.

HOW TO CUSTOMISE
----------------
Open index.html in any text editor (Notepad, VS Code, etc.) and find
the placeholder text below. Replace each one with your real details:

  "Your Name"                  → Your full name (appears in sidebar)
  "Financial Services"         → Your job title or specialisation
  "Senior Financial Analyst"   → Your actual job titles
  "Global Financial Services"  → Your actual employers
  "2021 — Present"             → Your actual dates
  "your.email@example.com"     → Your real email address
  "linkedin.com/in/yourprofile"→ Your LinkedIn URL
  "+44 (0) XXX XXX XXXX"       → Your phone number
  "London, United Kingdom"     → Your location
  The stats on the home page (8+, £2B, 3) → Your own numbers
  © 2024                       → Current year

HOW TO PUBLISH (FREE OPTIONS)
------------------------------

OPTION 1 — Netlify Drop (Easiest, ~1 minute)
  1. Go to: https://app.netlify.com/drop
  2. Drag and drop the entire folder containing all 4 files.
  3. You'll get a free public URL instantly (e.g. bright-fox-123.netlify.app).
  4. Optional: claim a free custom subdomain in the Netlify dashboard.

OPTION 2 — GitHub Pages (Free, professional URL)
  1. Create a free account at https://github.com
  2. Create a new repository named: yourusername.github.io
  3. Upload all four files to the repository.
  4. Go to Settings → Pages → set source to "main" branch.
  5. Your site will be live at: https://yourusername.github.io

OPTION 3 — Vercel (Free, fast)
  1. Go to: https://vercel.com and sign up.
  2. Click "Add New Project" → "Browse" → upload the folder.
  3. Deploy — you'll get a free URL (e.g. portfolio-xyz.vercel.app).

TIPS
----
- All four files must stay in the same folder when you upload them.
- The site uses Google Fonts (Cormorant Garamond + Montserrat).
  An internet connection is needed to load the fonts correctly.
- The site is fully responsive and works on mobile, tablet, and desktop.
